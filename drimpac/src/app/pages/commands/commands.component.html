Commands

Publish Congestion Points Publish Congestion Points, including their associated Connections, to the Common Reference, in order to make them available to the AGRs.

 Retrieve Active Aggregators Retrieves a list of all DSO-registered Congestion Points with, for each such point, a list of Aggregators representing Prosumers there, including the number of Connections each Aggregator represents.

Create Connection Forecast Event Create forecast for connections which are not served by an AGR. The DSO must create the non-aggregator connection forecast before it receives any D-Prognoses from AGRs. Therefore the DSO connection forecast must be called earlier than the AGRs connection forecast.
